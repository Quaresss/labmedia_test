# Тестовое задание для компании Labmedia 

Одностраничное приложение на **Vue 3** и **TypeScript**: демонстрация универсальных composables для валидации форм и HTTP-запросов, UI на **PrimeVue**.

## Тестовое API

Для примеров HTTP используется публичное **JSONPlaceholder**: `https://jsonplaceholder.typicode.com`.
